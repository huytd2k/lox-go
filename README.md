# golox
Golang implement for `lox` programming language 

More detail: [craftinginterpreters](https://craftinginterpreters.com/introduction.html)

Current implement process: 3.1
## prerequisites
- golang >= 1.16

## build
```bash
$ make build
```

## run
```bash
# Run in REPL mode
$ bin/lox 
# Run from source
$ bin/lox helloworld.lox
```
