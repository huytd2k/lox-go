### golox
Golang implement for `lox` programming language 
More detail: [craftinginterpreters](craftinginterpreters.com/introduction.html)
## prerequisites
- golang >= 1.16

## build
```bash
$ make build
```

## run
```bash
# Run in REPL mode
$ /bin/lox 
# Run from source
$ /bin/lox helloworld.lox
```
